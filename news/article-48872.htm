Windows 系统电脑如何下载安装并运行 Kubectl 查看云服务器 k8s 集群资源

  • 首页

官方文档：https://www.kubernetes.org.cn/installkubectl

1、Buckectl 下载

win + R 打开 cmd 执行命令下载kubectl.exe 文件

curl -LO https://storage.googleapis.com/kubernetes-release/release/v1.7.0/bin/windows/amd64/kubectl.exe

2、配置环境变量

增加 kubectl.exe 到环境变量

3、配置 kubectl

在 C 盘用户文件下创建 .kube 文件，创建 config 文件

最后编辑 config 文件，把集群信息粘贴到 config 文件中保存。在 cmd 中运行 kubectl version 显示 client 和 server 信息说明成功。
